# RohitPortfolio

Netflix-style Blazor WebAssembly portfolio site for Venkata Satya Sai Rohit Kuppili.

## Requirements

- .NET 8 SDK

## Run locally

```bash
dotnet restore
dotnet run
```

Then open the URL shown in the console (usually https://localhost:7224 or similar).
